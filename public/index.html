Welcome!

The Milan Festival of Performing Arts takes place in September each year. We organize and promote various kinds of events across the area of Milan: music, dances, theatrical entertainment and seminars. This website provides you with all the information about each event, the artists involved and how to get tickets.

This year (2019), the festival lasts from the 5th to the 18th of September.

We hope to see you soon!

In case this wasn't clear, this is not a real event. This website is a university project with the purpose of practising html, javascript and css.
